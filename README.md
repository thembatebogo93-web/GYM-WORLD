GYM-WORLD
GYM-WORLD is a modern fitness subscription platform where creators upload workout videos and sell structured fitness programs. Users can subscribe to creators, access premium content, and follow guided training plans. Built with Next.js for performance, scalability, and clean user experience.

Features
Creator profiles with subscription pricing  
Video upload and streaming  
Monthly subscription system in South African Rands (ZAR)  
User authentication and dashboard  
Glassmorphic design with navy and metallic blue theme  

Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js with Express  
Database: SQLite  
Authentication: JWT  
Payments: PayFast (South Africa)  

Status
Live demo available. Backend running on port 5000. Frontend serving on port 3000.

Folder Structure
Backend – server.js, database, API routes  
Frontend – HTML pages, CSS styles, client-side JavaScript  

Setup
Backend: cd Backend, npm install, node server.js  
Frontend: npx serve . , open http://localhost:3000
